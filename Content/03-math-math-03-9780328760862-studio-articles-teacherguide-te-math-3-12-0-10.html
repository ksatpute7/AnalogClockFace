LESSON 10: How Many Pies?

Lesson Overview

Students will write an equation to represent a two-step problem. In the equation, the unknown may not be isolated on one side of the equation.

Content Standards

3.OA—Operations and Algebraic Thinking

Solve problems involving the four operations, and identify and explain patterns in arithmetic.

8. Solve two-step word problems using the four operations. Represent these problems using equations with a letter standing for the unknown quantity. Assess the reasonableness of answers using mental computation and estimation strategies including rounding.

Key Concepts

In this lesson, students solve this problem:

Jose’s father bought a case of 100 apples. They made some apple pies. They used 7 apples for each pie. When they were done, they had 58 apples left. How many pies did they bake?

One possible word equation is:

number of apples used for pie + number of apples left = number of apples in the case

If n is the number of pies made, then the number of apples used for all of the pies is n × 7. The equation is:

n × 7 + 58 = 100

Students can solve this equation by using the relationship between addition and subtraction to get n × 7 = 100 − 58 = 42. They can complete the solution by using mental math or by using the inverse relationship between multiplication and division to get n = 42 ÷ 7 = 6.

Two possible word equations involving subtraction are:

number of apples in the case – number of apples used for pie = number of apples left and

number of apples in the case – number of apples left = number of apples used for pie

The respective equations in these cases are:

100 − n × 7 = 58 and 100 – 58 = n × 7

The first can be solved by using the relationship between addition and subtraction to get 100 – 58 = n × 7, which gives 42 = n × 7. This equation can then be solved by using mental math or the relationship between multiplication and division.

To solve the second equation, students can subtract to get 42 = n × 7 and then use mental math or the relationship between multiplication and division.

Students can also solve any of these equations by using a guess-and-check strategy.

Goals and Learning Objectives

  • Write equations to find unknown values in word problems

  • Express two-step problem situations using one equation

  • Write two-step equations that do not require parentheses due to the order of operations

  • Solve a simple equation in which the unknown is not alone on one side

  • Model problem situations

  • Estimate solutions

  • Check answers using the relationship between operations
