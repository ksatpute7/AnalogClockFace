Task 7: Skip-Counting Patterns

Lesson Guide

Have each student write a summary about skip-counting by 5s or 10s. Then have students work together to write a class summary. If the resulting class summary is helpful, share it with the students.

A Possible Summary

Skip-counting by 10s is like adding 10 each time. When you skip-count by 10s you move down the same column in the number chart.

Skip-counting by 5s is like adding 5 each time. When you skip-count by 5s you shade two columns that are 5 away from each other.

Additional Discussion Points

There are additional things you might want to discuss.

  • Skip-counting pattern when started with a number that does not have a 5 or 0 in the ones place.

  • Similarities and differences between counting forward and backward in the pattern.
