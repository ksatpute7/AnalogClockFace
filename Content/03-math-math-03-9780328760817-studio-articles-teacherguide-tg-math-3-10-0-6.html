LESSON 6: Line Plots

Lesson Overview

Students will interpret and construct line plots.

Content Standards

3.MD—Measurement and Data.

  • 4. Generate measurement data by measuring lengths using rulers marked with halves and fourths of an inch. Show the data by making a line plot, where the horizontal scale is marked off in appropriate units— whole numbers, halves, or quarters.

Key Concepts

Line plots look similar to picture graphs in that there is relationship between the icon and a data point. A line plot shows numerical data, while a bar graph or a picture graph shows categorical data. In a line plot, one icon (usually, a dot or an X) always represents one data point, while for a picture graph each icon can represent more than one data point.

The purpose of a line plot is to show the shape of the data, including clusters of values or gaps in the values. A line plot indicates which, if any, value or values are typical. The most common value in a data set is the value with the greatest number of icons above it. This value, called the mode, may be considered a typical value, especially if many other values in the data set are clustered around it. The halfway point of the data—that is, the value that half the data values are below and half are above—may also be considered a typical value. This value is called the median. The difference between the least and greatest values is called the range of the data. The range is an indication of how much the data are spread.

These characteristics can be observed informally and conclusions can be drawn about the data.

Goals and Learning Objectives

  • Interpret line plots

  • Construct line plots

Materials

  • Rulers, showing 1/4 inch, 1 per pair of students

  • Sheets of paper, 2 sheets per student
