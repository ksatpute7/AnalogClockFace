LESSON 8: Groups of Three

Lesson Overview

Students repeat the threes chant during the Opening. Be sure to find a few minutes outside of math class to have students recite the twos and fives chants as well.

Students learn that an array is an arrangement of objects in equal rows. They work with multiplication in the context of arrays. They consider what happens to the total number of objects in an array when rows are added or removed.

  • SWD:

    Struggling mathematicians may spend too much time sketching an array rather than thinking about the math. Consider providing grid paper so students can outline their array. If they are ready to move beyond outlining the array on the grid, think about introducing them to the open array. Open arrays may be helpful in avoiding sketches that often become cumbersome. Encourage students using the open array to proportionally set up the array until you are confident that the conceptual understanding is in place.

Key Concepts

A rectangular array is an arrangement of objects in rows and columns. Rows are horizontal arrangements of objects in straight lines. Columns are vertical arrangements of objects in straight lines. In an array, all rows have an equal number of objects in them, and all columns have an equal number of objects in them.

Arrays are introduced in this lesson for two reasons:

  • First, to continue developing students’ understanding that the multiplicand is a unit that can be counted (The rows in an array can be counted; all of the individual objects do not have to be counted.)

  • Second, to pave the way for beginning to investigate the commutative property (An array with 3 rows and 4 columns, 3 × 4, can be rotated 90 degrees so that it becomes an array with 4 rows and 3 columns, 4 × 3. The convention is that the first factor counts the number of rows and the second factor counts the number of columns.)

Goals and Learning Objectives

  • Learn the threes chant to help memorize the threes facts

  • Use an array to model and solve a problem involving rows of 3 stars

  • Examine the patterns in the Threes Multiplication Table
