LESSON 2: Choosing a Graph Project

Lesson Overview

Students will select a partner and then a question to investigate for your graphing data project.

Content Standards

3.MD—Measurement and Data

  • 3. Draw a scaled picture graph and a scaled bar graph to represent a data set with several categories. Solve one- and two-step “how many more” and “how many less” problems using information presented in scaled bar graphs. For example, draw a bar graph in which each square in the bar graph might represent 5 pets.

  • 4. Generate measurement data by measuring lengths using rulers marked with halves and fourths of an inch. Show the data by making a line plot, where the horizontal scale is marked off in appropriate units— whole numbers, halves, or quarters.

Key Concepts

The graphing data project is designed to allow students to interact with data and graphing in a deeper way. Students can explore graphs for categorical data (bar graphs and picture graphs) and graphs for numerical data (line plots). Some topics can be shown using any of the three graphs. For example, “How Many Cars Are at Home?” can be shown as categorical data, with the number of cars as the categories. A bar graph or picture graph could be used. The data is also numerical and a line pot could be used to show the typical number of cars.

Goals and Learning Objectives

  • Choose a partner for the graphing data project

  • Choose a question for the graphing data project
