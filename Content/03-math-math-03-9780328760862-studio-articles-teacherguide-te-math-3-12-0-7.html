LESSON 7: How Many Girls?

Lesson Overview

Students will write one equation that can be used to solve a two-step problem. Students will continue to explore the use of parentheses to indicate which calculation should be done first.

Content Standards

3.OA—Operations and Algebraic Thinking

Solve problems involving the four operations, and identify and explain patterns in arithmetic.

8. Solve two-step word problems using the four operations. Represent these problems using equations with a letter standing for the unknown quantity. Assess the reasonableness of answers using mental computation and estimation strategies including rounding.

Key Concepts

In this lesson, students solve another two-step problem:

There are 18 children at the park. Half of them are girls. Then 7 more girls come to the park. Now how many girls are at the park?

As in earlier lessons, students draw a diagram to model the problem and then represent the problem with a single equation. Here is one possible diagram:

Estimate: 20 ÷ 2 + 7= 17

18 ÷ 2 + 7= n

9 + 7= n

16= n

To solve this problem, students divide 18 by 2 to find the number of girls in the park to start. Then they add 7 to find the number of girls in the park now. This can be represented by:

  • 18 ÷ 2 + 7 = n

  • 7+18 ÷ 2 = n

  • 7 + (18 ÷ 2) = n

Note that the parentheses in the last problem are not needed. The order of operations states that division is done before addition, even if addition comes first when reading an equation from left to right.

Goals and Learning Objectives

  • Write equations to find unknown values in word problems

  • Express two-step problem situations using one equation

  • Write two-step equations that do not require parentheses due to the order of operations

  • Model problem situations

  • Estimate solutions

  • Check answers using the relationship between operations
