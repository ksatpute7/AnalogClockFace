LESSON 3: Picture Graphs

Lesson Overview

Students will interpret and construct picture graphs that involve scale.

Content Standards

3.MD—Measurement and Data

  • 3. Draw a scaled picture graph and a scaled bar graph to represent a data set with several categories. Solve one- and two-step “how many more” and “how many less” problems using information presented in scaled bar graphs. For example, draw a bar graph in which each square in the bar graph might represent 5 pets.

Key Concepts

Picture graphs sometimes show a one to one correspondence between the icons and the frequency. However, in a variation of scale, the icons can represent a quantity of more than one, which necessitates including a key on the graph. This idea ties to multiplication, as the icons for a category are groups of some quantity: the number of icons gives us the multiplier while the value of each icon gives us the multiplicand.

In a picture graph each icon represents a group of data points, unlike a bar graph where the scale indicates intervals with possible amounts in between. However, picture graphs, like bar graphs with larger scales, often show approximations of data, meant to display a comparison between categories, because a fraction of the icon can't be shown easily and data is not always in convenient multiples. It is understood that if an icon represents 1,000, then categories are shown to the nearest 1,000. Sometimes one half of an icon is used.

Goals and Learning Objectives

  • Construct a picture graph using a key

  • Interpret a picture graph
