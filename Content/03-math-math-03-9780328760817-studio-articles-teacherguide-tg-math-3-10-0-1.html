LESSON 1: How Fast is Fast?

Lesson Overview

Students will review how data is displayed in bar graphs. Students will then consider what to do when the data set is larger.

Content Standards

3.MD—Measurement and Data

  • 3. Draw a scaled picture graph and a scaled bar graph to represent a data set with several categories. Solve one- and two-step “how many more” and “how many less” problems using information presented in scaled bar graphs. For example, draw a bar graph in which each square in the bar graph might represent 5 pets.

Key Concepts

A bar graph is a visual display of data that shows quantitative comparisons for categories. The height of each bar indicates that category's frequency. Picture graphs can show the same information, but use individual icons to represent frequency. So far, students have seen only a one to one correspondence between the icons and their frequency in picture graphs and a scale with increments of 1 unit for bar graphs. Now students must begin to think about the possibility that the interval of a bar graph could be greater and that an icon in a pictograph can represent more than one.

This curiosity about what to do with more data sets up the purpose of the unit, which is to use scale to display sets of larger data using graphs students are already familiar with.

Goals and Learning Objectives

  • Interpret bar graphs and picture graphs

  • Think about how to display sets of larger data.
