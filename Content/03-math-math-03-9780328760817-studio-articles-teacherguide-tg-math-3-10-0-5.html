LESSON 5: Large Scale Bar Graphs

Lesson Overview

Students will interpret a bar graph with a scale of 100. Then you will construct a bar graph for a set of data with large numbers.

Content Standards

3.MD—Measurement and Data

  • 3. Draw a scaled picture graph and a scaled bar graph to represent a data set with several categories. Solve one- and two-step “how many more” and “how many less” problems using information presented in scaled bar graphs. For example, draw a bar graph in which each square in the bar graph might represent 5 pets.

Key Concepts

As discussed in the previous lesson, bar graphs are used to represent information. The categories, shown on the horizontal axis, show the information that is represented. The scale is used to show the amount that the data represent. Choosing an appropriate scale is important when constructing bar graphs for a set of data with large numbers.

Goals and Learning Objectives

  • Interpret bar graphs

  • Construct bar graphs for a set of data with large numbers
