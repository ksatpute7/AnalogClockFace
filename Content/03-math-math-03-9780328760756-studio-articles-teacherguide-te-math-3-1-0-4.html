LESSON 4: What Time Will It Be?

Lesson Overview

You have found the amount of elapsed time before. Today you will find the start time or the end time when given a time and the number of minutes that have gone by.

Key Concepts

Previously students have solved problems in which they were given a start time and an end time and they had to find the elapsed time:

Start time: given

End time: given

Elapsed time: unknown

Now students must use what they know to solve problems where the start time (or end time) and the elapsed time are given, and they must find the end time (or start time):

Start time: given or Start time: unknown

End time: unknown End time: given

Elapsed time: given Elapsed time: given

Goals and Learning Objectives

  • Determine the time a certain number of minutes after a given time

  • Determine the time a certain number of minutes before a given time

  • Solve problems involving elapsed time when the time interval and either the start or the end time are given

  • Solve problems involving elapsed time using a calculation rather than counting the tick marks on a timeline

Materials
