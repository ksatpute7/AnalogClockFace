LESSON 4: Bar Graphs

Lesson Overview

Students will explore how to construct a bar graph for a large set of data where a scale with increments greater than 1 is needed. Students will also interpret bar graphs.

Content Standards

3.MD—Measurement and Data

  • 3. Draw a scaled picture graph and a scaled bar graph to represent a data set with several categories. Solve one- and two-step “how many more” and “how many less” problems using information presented in scaled bar graphs. For example, draw a bar graph in which each square in the bar graph might represent 5 pets.

Key Concepts

A bar graph is partly about ratio and proportion. The ratio of the lengths of two bars indicates the ratio of the frequencies they represent. For example, a bar that is three times the length of another bar represents a frequency value that is three times as great. This will be true regardless of the scale of the graph.

At this grade, scale can be thought of as shrinking the graph so that the space between each number is too small to label, so every fifth number (for example) is shown. The height of bars between these numbers is estimated to a certain degree, depending on the scale.

Goals and Learning Objectives

  • Interpret bar graphs

  • Construct bar graphs for a large set of data using a scale
